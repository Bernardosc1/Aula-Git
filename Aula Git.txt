Git init - Inicializa o Git 

Git add - Adiciona arquivos para serem tracked

Git Branch -M {name} - Define a branch utilizada no commit

Git Status - Mostra os status da app

Git Commit -m {message} - Identificando e registrando commit

Git Restore {arquivo} - Volta o código ate o ponto do ultimo commit

Git log - Mostra um Log dos commits anteriores

Git Checkout {id do commit} - Permite voltar em um commit passado

Git Checkout main - Retorna para commit mais recente

Git remote add origin {url do repositório} - Define o caminho do repositório remoto

Git push -o origin {nome da branch} - Joga os commits do repositório local para o remoto

Git pull - Traz as modificações do repositório remoto até o local
